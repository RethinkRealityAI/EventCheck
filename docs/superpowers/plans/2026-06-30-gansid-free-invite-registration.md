# GANSID Free-Invite Registration + Contacts Admin Polish — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give the bulk-contact invite path a dedicated, stripped-down, single-page FREE registration form with locked prefill and a ticket-first → optional-account flow; plus a manual add-contact action, hardened add-registrant modal, a responsive invite modal, and "GANSID Congress 2026" email branding with a bluer gradient.

**Architecture:** The new form is a GANSID **data row** (idempotent seed), not schema. The invite flow already renders `renderMode:'single'` one-page forms and shows a ticket-first success screen, so the code changes are small and surgical: lock prefilled inputs in `FormRenderer`, default the invite picker, move the account opt-in onto the success screen and defer the `contact-invite-account` call. The follow-ups (add-contact, modal hardening, responsive, branding) are isolated, independently-shippable phases. Pure helpers get vitest tests; UI is verified live in the preview (admin is logged in).

**Tech Stack:** React 18 + TS + Vite + Tailwind v4, Supabase (Postgres + edge functions in Deno), vitest, Supabase CLI (GANSID is CLI-only).

**Spec:** [docs/superpowers/specs/2026-06-30-gansid-free-invite-registration-design.md](../specs/2026-06-30-gansid-free-invite-registration-design.md)

```js
// scripts/seed-gansid-invite-form.mjs
// Usage: SVC_KEY=<gansid service_role> node scripts/seed-gansid-invite-form.mjs
import fs from 'node:fs';
const GAN = 'https://gticuvgclbvhwvpzkuez.supabase.co';
const SVC = process.env.SVC_KEY;
if (!SVC) { console.error('Set SVC_KEY (GANSID service_role)'); process.exit(1); }
const DROP = new Set(['f_mode', 'f_present', 'f_ticket']);
const KEEP_REQUIRED = new Set(['f_consent_photo', 'f_consent_terms', 'f_consent_liability']);

const r = await fetch(`${GAN}/rest/v1/forms?id=eq.gansid-congress-2026&select=fields`, {
  headers: { apikey: SVC, Authorization: `Bearer ${SVC}` },
});
const [src] = await r.json();
const fields = src.fields
  .filter(f => !DROP.has(f.id))
  .map(f => ({ ...f, required: KEEP_REQUIRED.has(f.id) ? true : false }));

const settings = {
  renderMode: 'single',
  successTitle: "You're registered 🎟️",
  submitButtonText: 'Complete my free registration',
};
const row = {
  id: 'gansid-congress-2026-invite',
  title: 'GANSID Congress 2026 — Free Registration',
  description: 'Confirm your free registration for the GANSID Congress 2026.',
  form_type: 'event',
  status: 'active',
  show_in_portal: false,
  fields,
  settings,
};
const lit = (o) => `'${JSON.stringify(o).replace(/'/g, "''")}'::jsonb`;
const sql = `-- Generated by scripts/seed-gansid-invite-form.mjs. Idempotent upsert.
insert into public.forms (id, title, description, form_type, status, show_in_portal, fields, settings)
values ('${row.id}', '${row.title.replace(/'/g, "''")}', '${row.description.replace(/'/g, "''")}',
        '${row.form_type}', '${row.status}', ${row.show_in_portal}, ${lit(fields)}, ${lit(settings)})
on conflict (id) do update set
  title = excluded.title, description = excluded.description, form_type = excluded.form_type,
  status = excluded.status, show_in_portal = excluded.show_in_portal,
  fields = excluded.fields, settings = excluded.settings;
`;
fs.mkdirSync('supabase/seeds', { recursive: true });
fs.writeFileSync('supabase/seeds/gansid-invite-form.sql', sql, 'utf8');
console.log(`Wrote supabase/seeds/gansid-invite-form.sql (${fields.length} fields, required: ${fields.filter(f=>f.required).map(f=>f.id).join(',')})`);
```

```ts
export function normalizeContactInput(input: { name: string; email: string; tags?: string[] }) {
  return { name: input.name.trim(), email: input.email.trim().toLowerCase(), tags: input.tags ?? [] };
}
export function isDuplicateEmail(email: string, rows: { email: string }[]) {
  const e = email.trim().toLowerCase();
  return rows.some(r => (r.email || '').trim().toLowerCase() === e);
}
export async function createImportedContact(input: { name: string; email: string; tags?: string[] }) {
  const n = normalizeContactInput(input);
  const { data, error } = await supabase.from('imported_contacts')
    .insert({ name: n.name, email: n.email, tags: n.tags, email_status: 'pending' })
    .select('id'); // rowcount check per rule #4
  if (error) throw error;
  if (!data || data.length === 0) throw new Error('Contact was not created (0 rows).');
  return data[0].id as string;
}
```

```ts
export function isValidEmail(email: string): boolean {
  return /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test((email || '').trim());
}
export function validateRequiredAnswers(
  fields: { id: string; required?: boolean }[],
  answers: Record<string, any>,
): string[] {
  return fields.filter(f => f.required).filter(f => {
    const v = answers[f.id];
    return v === undefined || v === null || (typeof v === 'string' && v.trim() === '') ||
      (Array.isArray(v) && v.length === 0) || v === false;
  }).map(f => f.id);
}
```
